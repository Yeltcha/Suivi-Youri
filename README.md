# TRAINING by Yeltcha

Application de suivi d’entraînements de bodybuilding, installable sur téléphone et synchronisable avec Supabase.

La version 1.7.0 ajoute les programmes entièrement libres, le RIR désactivable exercice par exercice, la validation et le verrouillage des séries, le tonnage par séance du programme actif et la correction de la durée enregistrée. Elle restaure aussi les icônes nécessaires à l’installation PWA.

Voir [README_INSTALLATION.md](README_INSTALLATION.md) pour la mise en ligne sur GitHub Pages et l’installation sur téléphone.
