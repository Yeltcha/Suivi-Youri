# BODYSSEUS

**Forge Your Odyssey.**

Application de suivi d’entraînements de bodybuilding, installable sur téléphone et synchronisable avec Supabase.

La version 1.8.0 adopte l’identité BODYSSEUS, ajoute le détail complet et non éditable des séances enregistrées, et transforme le logo en retour universel vers l’accueil. Si une série non validée ou une modification de séance modèle est en cours, l’utilisateur choisit d’abord de conserver, d’abandonner ou de poursuivre l’édition. Les icônes PWA BODYSSEUS sont incluses dans le dépôt.

Voir [README_INSTALLATION.md](README_INSTALLATION.md) pour la mise en ligne sur GitHub Pages et l’installation sur téléphone.
