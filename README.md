# BODYSSEUS

**Forge Your Odyssey.**

Application de suivi d’entraînements de bodybuilding, installable sur téléphone et synchronisable avec Supabase.

La version 1.9.0 adopte la palette « Bronze homérique », retire les recommandations générales d’échauffement des séances modèles et centralise l’historique complet sur l’accueil. Les séries d’échauffement restent disponibles individuellement avec le bouton `W`. Le cache PWA v12 et les icônes BODYSSEUS sont inclus dans le dépôt.

Voir [README_INSTALLATION.md](README_INSTALLATION.md) pour la mise en ligne sur GitHub Pages et l’installation sur téléphone.
